# How to Use This Book

*(Front matter — to be finalized by the writing workflow.)*

This book is a complete, self-contained study companion for **SOA/CAS Exam P (Exam 1)**, told as the story of Ash Ketchum's journey across the Kanto region. The narrative is the delivery vehicle; the probability curriculum is the cargo, and the cargo is complete.

**The promise.** Work every chapter, every worked example, and every problem set — and drill practice software (Coaching Actuaries / ADAPT) in parallel — and you should sit Exam P expecting a 9 or 10. Target benchmarks, stated throughout: **24/30** on practice exams, **Earned Level 7+**, **6 minutes per question**.

**The exam, in brief.** 3 hours · 30 multiple-choice questions (A–E) · computer-based · scored 0–10 (6 passes) · **no penalty for wrong answers — never leave a blank.** A normal table is provided on-screen. Calculator: TI-30XS MultiView (recommended) or BA II Plus, memory cleared.

**How each chapter is built.** Every chapter follows the same ten-part skeleton: a Cold Open episode → Oak's Briefing (learning outcomes) → Pokédex Entries (theory) → Worked Examples → Trainer's Tips → Team Rocket's Trap → the Gym Battle capstone → the Gym Challenge problem set (Route Trainers → Gym Battles → Elite Challenge) → full Answer Key → the Badge Earned mastery checklist.

**The boxes.** Watch for the colored callouts: **Pokédex Entries** (theory), **Trainer's Tips** (exam craft), **Team Rocket's Trap** (the canonical mistake, diagnosed), and **⬛ FROM KANTO TO THE REAL WORLD** (why this matters for the actuarial career the exam unlocks).
